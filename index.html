Explicação sobre o problema do CORS:
O erro que você está enfrentando está relacionado ao Cross-Origin Resource Sharing (CORS), uma política de segurança do navegador que impede que páginas web façam solicitações a um domínio diferente daquele que serviu a página. O CORS Anywhere é um serviço que permite que seu navegador acesse recursos de outros domínios como se estivessem no mesmo domínio. Para utilizá-lo, é necessário desbloquear o proxy clicando no link fornecido, que autoriza temporariamente o uso do serviço para seu navegador.
